# offline development
For local offline development, run `serverless offline start`, which will fire an init and a end lifecycle hook which is needed for `serverless-offline` and `serverless-dynamodb-local` to switch on/off resources.

## local HTTP API endpoint will be available on http://localhost:3000.
## local Websocket API endpoint will be available on http://localhost:3001.
## local DynamoDB Shell will be available on http://localhost:8000/shell.
